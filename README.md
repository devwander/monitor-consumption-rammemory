# RAM consumption monitor

The aim of this project was to create an endpoint so that you could access system ram data.

---
## Technology
- Node.js

---
## Get Start

```bash
git clone https://github.com/devwander/monitor-consumption-rammemory

# NPM
npm install

# Yarn
yarn install
```

---

## Startup

```bash
# NPM
npm start

# Yarn
yarn start
```
